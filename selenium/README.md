# Selenium UI Testing Setup

## Prerequisites
- Python 3.x
- Install dependencies:
  ```bash
  pip install selenium
  ```
- Download [ChromeDriver](https://chromedriver.chromium.org/downloads) and add it to your PATH.

## How to Run
1. Start your frontend server (Vite/React) so the app is available.
2. Run the test:
   ```bash
   python selenium/test_login.py
   ```

## Example Test
See `selenium/test_login.py` for a sample login UI test.
